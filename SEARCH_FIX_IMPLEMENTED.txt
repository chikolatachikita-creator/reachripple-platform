## TROUBLESHOOTING

**Dropdown not showing?**
→ Check z-index is 9999, not 60
→ Clear localStorage: `localStorage.clear()`
→ Hard refresh: Ctrl+Shift+R

**Only showing 1-3 results?**
→ Backend not recompiled. Run: `npx tsc`
→ Restart backend: Stop and run `npm run dev` again

**API error in console?**
→ Visit http://localhost:3001/api/location/location-suggest?q=N1 directly
→ Should return JSON with results array
→ Check backend is running on port 3001

**Keyboard navigation not working?**
→ Click in search box first to focus it
→ Use ↑↓ arrows to navigate dropdown
→ Press Enter to select highlighted item
