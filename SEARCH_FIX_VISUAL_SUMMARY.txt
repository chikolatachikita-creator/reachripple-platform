🐛 TROUBLESHOOTING

Problem: Dropdown still not visible
Solution: 
  • Clear browser cache (Ctrl+Shift+Delete)
  • Hard refresh (Ctrl+Shift+R)
  • Check z-index is 9999 (not 60)
  • Restart backend

Problem: Only showing 1-3 results
Solution:
  • Backend may not have recompiled
  • Run: npx tsc
  • Restart: npm run dev
  • Check /api/location/location-suggest?q=N1 returns full results

Problem: Search box doesn't update URL
Solution:
  • Make sure you SELECTED an option (not just typed)
  • Try pressing Enter key instead of clicking
  • Check browser console (F12) for errors
